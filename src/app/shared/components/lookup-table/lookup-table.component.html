Use case: you have a value that has to be interpolated according to another value, if a switch case would be a solution, then this is a solution. Eg:
  • translating an id to it's meaning
  • translating a string to it's illustrating icon
But depending on how the lookup variable is set up ( enums, records, objects, maps ). The template syntax might give you grief
With a plain object: @for (arrow of toMatch; track $index) {
{{ arrow }} =
}
With an enum: @for (arrow of toMatch; track $index) {
{{ arrow }} =
}
Without the pipe @for (arrow of toMatch; track $index) { }
